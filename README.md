# Medical Worker Forms

This project is a simple web application developed using HTML, CSS, and JavaScript. It contains two forms: a Medical Expenses Form and a Worker Progress Report. JavaScript is used to load the data dynamically into the HTML pages.

## Features

### Medical Expenses Form
- Displays worker and claim information.
- Shows treatment and medical expense details.
- Clean and responsive user interface.

### Worker Progress Report
- Displays return-to-work details.
- Shows recovery progress and comments.
- Displays medical treatment information.
- Includes additional information, Worker ID, and Submitted Date.

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How It Works

- HTML is used to create the structure of the forms.
- CSS is used to design and style the pages.
- JavaScript stores the form data in objects and displays it dynamically using DOM manipulation.
